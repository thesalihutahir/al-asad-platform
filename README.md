# Al-Asad Education Foundation Platform

Official web platform for **Al-Asad Education Foundation**, an Islamic educational and community-focused organisation led by **Sheikh Goni Dr. Muneer Ja'afar Katsina**.

The platform presents the foundation’s mission, programs, media, and ways for supporters to engage, volunteer, partner, and donate.

---

## What you can do on the platform

### Explore
- Learn about the foundation and its leadership
- View programs and initiatives
- Read updates, articles, and research publications
- Access media resources:
  - Videos
  - Audios
  - Podcasts
  - eBooks
  - Photo gallery

### Get involved
- Donate to support active causes
- Volunteer
- Partner with the foundation
- Send messages through the contact page

---

## Admin system (internal)

The platform includes a secure admin area for authorised staff to:
- Publish and manage content
- Upload and organise media
- Manage programs
- Review submissions (volunteers and partners)
- Manage donations and confirmations
- Maintain basic site settings

Admin access is protected and role-based.

---

## Technology (high level)

- Next.js (web application framework)
- Tailwind CSS (styling)
- Firebase (database, authentication, storage)
- Paystack (online donations)

---

## Deployment

The platform is deployed using modern cloud hosting with secure environment configuration.
